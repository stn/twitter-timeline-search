# twitter-timeline-search
Search your Twitter timeline


## Preliminary

A Twitter API tokens is needed for now.

## Install

```bash
git clone https://github.com/stn/twitter-timeline-search.git
cd twitter-timeline-search
```

for Pipenv users:

```bash
pipenv --python 3
pipenv install
pipenv shell
```
